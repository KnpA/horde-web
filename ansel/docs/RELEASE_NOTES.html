Release notes for the latest release

The Horde Team is pleased to announce the final release of the Ansel Photo 
Management and Web Gallery application version H3 (1.1).

Ansel is a full featured photo management application. With it, you can organize
your photos in any number of galleries and subgalleries, share galleries among
other Horde users or even make them public. Ansel supports a wide range of
features. See http://www.horde.org/ansel/ for more information.

The major changes compared to the Ansel H3 (1.0) versions are:
    * Support for geotagging, and the integration of Google Maps.
    * Fixed a bug relating to the generation of screen images with the GD
      Horde_Image driver.
    * Fixed a bug relating to gallery view modes being reset to Normal when
      uploading images.
    * Performance improvements and bug fixes specific to issues with IE.
    * Numerous other bugfixes and improvements.
