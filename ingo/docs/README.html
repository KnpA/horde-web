Ingo is an email filter rules manager. It can generate Sieve and procmail scripts and upload them to the server (using a timsieved or VFS FTP driver, respectively). It can also create and execute IMAP commands to run filter rules.

This software is OSI Certified Open Source Software. OSI Certified is a certification mark of the Open Source Initiative.
