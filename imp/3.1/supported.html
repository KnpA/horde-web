PHP

  • Known Working
    • NOTE that PHP versions before 4.1.2 have a serious security vulnerability in the upload code. You should apply the patch or upgrade to 4.1.2 or later as soon as possible.
    • PHP 4.1.0 and later
  • Does Not Work
    • PHP version 4.2.3 has a bug in the IMAP code. You should use 4.2.2 or 4.3.x instead; see IMP mailing list archive
    • Problems being reported with University of Washington c-client imap-2001a; see IMP mailing list archive
    • PHP 3.x
    • PHP 4.0.x
