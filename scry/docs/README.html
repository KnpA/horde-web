Scry Version 0.1

What is Scry?

Scry is the Horde polls application, designed to be integrated with other Horde applications to provide polling/voting for users throughout the Horde suite.

This software is OSI Certified Open Source Software. OSI Certified is a certification mark of the Open Source Initiative.

Obtaining Scry

Further information on Scry and the latest version can be obtained at

http://www.horde.org/scry/
