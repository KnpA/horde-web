What is Nag?

Nag is a set of PHP scripts that implement a simple, multiuser task list manager. It makes extensive use of the Horde web application framework. The development versions also support multiple task lists and sharing task lists between users.

What does it require?

  • Absolute requirements: A web server running a recent version of PHP with support for a database with a PEAR driver, and recent (CVS or snapshot) version of PEAR.
  • More or less absolute: A browser capable of a reasonable level of JavaScript and tables.
