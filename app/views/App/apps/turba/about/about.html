What is Turba?

Turba is a set of PHP scripts that implement a contacts directory and management interface. Turba also provides an API for external applications to interface with address books, providing adding and searching features that you can hook right into other applications.

What does it require?

Absolute requirements: A web server running a recent PHP4 with support for either LDAP or a database with a PEAR driver, and recent (CVS or snapshot) version of PEAR.

More or less absolute (ie, you'd have to do serious code revision to get by without): A browser capable of a reasonable level of javascript and tables.

Can I try it?

There is no demo available yet.

Where can I see the code?

You can browse the current source tree online. Chora, Horde's web-based cvs frontend written in PHP, lets you go through the source tree and view the current code, as well as what's changed from version to version.
