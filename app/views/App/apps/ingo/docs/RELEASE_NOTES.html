Release notes for the latest release

The Horde Team is pleased to announce the final release of the Ingo Email
Filter Rules Manager version H5 (3.0.0).

Ingo is an email-filter management application. It is fully internationalized,
integrated with Horde and the IMP Webmail client, and supports both server-side
(Sieve, Procmail, Maildrop) and client-side (IMAP) message filtering. For more
information on Ingo, visit http://www.horde.org/apps/ingo.

The major changes compared to the Ingo version H4 are:
    * Completely redesigned user interface.
    * Added initial implementation of a smartphone view.
    * Added placeholder variables to vacation messages.
    * Added hook to customize authentication parameters.
    * Small improvements.
