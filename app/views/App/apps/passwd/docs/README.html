Passwd is the Horde password changing application. Right now, Passwd provides fairly complete support for changing passwords via Poppassd, LDAP, Unix expect scripts, the Unix smbpasswd command for SMB/CIFS passwords, Kolab, ADSI, Pine, Serv-U FTP, VMailMgr, vpopmail, and SQL passwords.

This software is OSI Certified Open Source Software. OSI Certified is a certification mark of the Open Source Initiative.
