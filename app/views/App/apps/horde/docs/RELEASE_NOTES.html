Release notes for the latest release

The Horde Team is pleased to announce the final release of the Horde
Application Framework version 3.3.11.

The Horde Application Framework is a modular, general-purpose web application
framework written in PHP.  It provides an extensive array of classes that are
targeted at the common problems and tasks involved in developing modern web
applications.

The major changes compared to Horde version 3.3.10 are:
    * Fixed XSS vulnerability when viewing details of a vCard.
    * Fix exporting recurrence exceptions to vCalendar 1.0.
    * Various fixed relating to synchronization.
    * Minor bug fixes and improvements.
