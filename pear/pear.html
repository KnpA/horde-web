Note: Do NOT use the PEAR packages provided on this page for Horde 3.0. Horde 3.0 requires PHP 4.3.0 that comes with a recent enough PEAR version. Use PEAR's command line tool instead to update or install required PEAR packages.

Horde 2.x, IMP 3.x, and all other Horde applications make use of the PEAR libraries to handle such tasks as database abstraction, mail delivery and logging. These libraries come bundled with PHP4. However, since PEAR is a rapidly evolving target, not everyone who has a workable version of PHP4 has a recent enough PEAR installation.

To make it easier for users to obtain the latest PEAR libraries, we have provided an archive containing a version of PEAR which will work with the latest Horde code. The package is available on our ftp site:
.tar.gz: /pear/pear-1.3.tar.gz
.tar.bz2: /pear/pear-1.3.tar.bz2
.zip: /pear/pear-1.3.zip

Since PEAR 1.3 (that is included in our pear-1.3 packages) a PHP version of 4.2 or higher is required. If you have an older PHP version, you can try an older PEAR package like pear-1.1.

To install this package simply backup your current /usr/local/lib/php (or wherever PEAR lives on your system; /usr/local/share/php/pear is another possibility) directory, and uncompress the contents of the archive above into that same directory.

Note that you DO NOT need to recompile PHP to install these libraries. Simply moving the new PEAR files into place will be sufficient; they are just PHP scripts and no recompilation or restarting of the server is necessary.
