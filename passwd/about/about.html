What is Password?

Password is a set of PHP scripts which implements methods to change user's passwords. It currently has support for changing passwords via a poppassd, ldap, smbpasswd, servuftp, and sql, vmailmgr, vpopmail, and expect script drivers.

Where can I see the code?

You can browse the current source tree online. Chora, Horde's web-based cvs frontend written in PHP, lets you go through the source tree and view the current code, as well as what's changed from version to version.
