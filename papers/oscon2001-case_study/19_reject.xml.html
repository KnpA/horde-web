Rejecting Features

  • Someone contributed a patch to let IMP 2.3 look basically like Outlook – frames, etc.
  • Very attractive to power users.
  • However, IMP's main installation base is universities, and thus the primary user base consists of average students.
  • Also, having a user preference which changes the entire user interface makes support and manuals difficult.
  • So, the patch was rejected.
