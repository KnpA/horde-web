Over-design: Turba

  • Horde's address book / directory program has an elegant object model which has eventually turned out to work quite nicely.
  • But for months, all that was there was the object model - NO WORKING CODE!
  • Something needs to exist and work in order to attract users/developers.
  • A less elegant but working prototype, or a part of the object model fully implemented, would have helped get things started.
