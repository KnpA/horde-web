More Registry Paths


The Registry is the central repository of path information for all installed Horde applications. We use this information in a number of ways. One example is to determine where templates live:

<?php require $registry->getTemplatePath() . '/common-header.inc'; ?>

All calls to the $registry object assume that you are asking for information about the application that is currently running. You can pass in another application as an argument, however:

<?php echo "IMP's templates are in: " . $registry->getTemplatePath('imp'); ?>
