URL generation


<?php

// A link to a script in the same app
$url = Horde::applicationUrl('foo.php');

// A full URL for a Location: header
$url = Horde::applicationUrl('foo.php', true);

// A link to another app
$url = Horde::url('bar.php');

?>
  
  • Adds the session ID to the URL if necessary
  • Horde::applicationUrl() generates a full relative path to the app
  • When full URLs are generated, http/https is used depending on $conf['use_ssl']
